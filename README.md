# ffmtools

> ffmtools is a package with multiple ffmpeg commands and child spawners, to help you do some magic and tricks with ffmpeg in your code!

## Table of Contents
- [Installation](#installation)
- [Features](#features)
- [Documents](#documents)
- [Examples](#examples)

### Installation
- Using NPM

```
npm install ffmtools
```

- Using yarn

```
yarn add ffmtools
```
